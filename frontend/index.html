Instructions for first time users:

To view any information, add /api/register at the end of the link and do a POST request with first name
last name, email (has to be unique per account) and password for login.
Then change register to login in the url and do a POST request with email and password to get your API key.

Using your API key in the header, you can then start CRUD operations via POST, GET, PUT and DELETE requests.
It is advisable to use Postman for these operations, you can find Postman here: https://www.getpostman.com/

Here are some of inputs for the api:
To get all the studentenhuizen: /api/studentenhuis
To get a specific studentenhuis by id: /api/studentenhuis/:id (e.g. /api/studentenhuis/1)
To get a more specific piece of data such as a deelnemer to a specific maaltijd from a specific studentenhuis: /api/studentenhuis/:id/maaltijd/:id/deelnemers
